The 5-layer architecture that solves all 8 gaps at once

The 8 Gaps (Validated by Wesco/Anixter Fortune 500)

👤

1. No ownership attribution

Agents have no tie to a named human. When the human leaves, the agent goes rogue with live credentials.

🔒

2. No scope enforcement

Agents routinely receive entitlements beyond their creator's access level.

⚠️

3. No SoD for agents

Toxic action combinations that would be blocked for humans are trivial for agents.

👁️

4. No app-owner approval

ServiceNow, Oracle, SAP, Salesforce owners have zero visibility or veto on agents calling their systems.

♻️

5. No lifecycle management

No provisioning workflow, no recertification, no HR-tied deprovisioning. Agents accumulate forever.

🛡️

6. No runtime enforcement

Discovery tools exist. Inline gateway checking every tool call against policy is unsolved at scale.

🚫

7. No data exfiltration guardrails

Agents can scoop PII/PHI/PCI far beyond intended scope.

📋

8. Audit trail gaps

Even Microsoft Copilot has documented cases bypassing M365 logs when acting on behalf of users.

The 5-Layer Solution

L1

Shadow Org Graph

Ingest HRIS â†’ live human graph â†’ hang agents as child nodes. HR termination = cascade deprovision of entire agent subtree in <60 seconds.

L2

Credential Clone Engine

Auto-issue scoped OAuth 2.1 credentials via Token Exchange (RFC 8693) against parent's live entitlements. Agent cannot exceed parent's access. Never copies â€” always derivatives.

L3

MCP-Aware API Brokerage Gateway

Sits between every agent and every target system. Intercepts every tool call, verifies parent scope, evaluates SoD/DLP, passes/blocks/redacts/escalates. <5ms p99 added latency target.

L5
